Requires webserver and MySQL database with hard coded username 'root' and no password (should be changed)

To run on xampp:
1. Download and install xampp https://www.apachefriends.org/index.html
2. Run xampp control panel and start Apache web server and MySQL database
4. Create folder in \xampp\htdocs
5. Put project files into your new folder \xampp\htdocs\YourFolder
6. Open web browser and go to "http://localhost/YourFolder"
7. Go to "http://localhost/phpmyadmin" to access database
